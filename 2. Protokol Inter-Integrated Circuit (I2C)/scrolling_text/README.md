<h1>Scrolling Text Dengan I2C</h1>

Selanjutnya, kita akan mencoba untuk menampilkan scrolling text. Scrolling text ini berguna ketika kita ingin menampilkan sebuah teks dengan panjang lebih dari 16 karakter, yang mana tidak dapat ditampilkan sekaligus pada layar LCD 16 x 2.

https://github.com/user-attachments/assets/adaaabcb-d4d3-4592-be0d-29c11f9fb2c4

Pada baris pertama layar LCD, kita akan menampilkan static text, sementara pada baris kedua, akan ditampilkan scrolling text, sehingga kamu dapat melihat perbedaan antara static text dengan scrolling text.
